# scf-auto-crosswalker

An AI-powered open-source tool designed to parse raw IT policies or Cloud Security Findings and automatically map them to the latest Secure Controls Framework (SCF) controls.

## Features
- **Auto-Updating SCF DB:** Directly fetches and parses the latest Secure Controls Framework data.
- **AI Mapping Engine:** Uses LLMs to analyze policies/findings and identify the corresponding SCF Control IDs with confidence scoring.
- **Lab Environment:** Includes sample data (`lab_data/`) so you can instantly test the tool's mapping capabilities without risking real company data.
